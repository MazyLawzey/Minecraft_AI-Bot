# 🤖 MINECRAFT AI BOT 1.3 - УЛУЧШЕННАЯ ВЕРСИЯ

## НОВЫЕ ВОЗМОЖНОСТИ УМНОГО БОТА

✨ ОСНОВНЫЕ ОСОБЕННОСТИ:

1. 🧠 ПРОГРЕССИВНЫЙ AI ДВИГАТЕЛЬ
   - Использует Ollama + GEMMA 3 для понимания русского языка
   - Распознаёт контекст и память о прошлых действиях
   - Саркастичный и живой характер
   - Понимает сложные команды и намерения

2. 🎯 АВТОНОМНОЕ ПОВЕДЕНИЕ
   - Бот может действовать самостоятельно без команд
   - Принимает стратегические решения
   - Приоритизирует: выживание > ресурсы > исследование

3. 💔 СИСТЕМА ВЫЖИВАНИЯ
   - Автоматическое управление здоровьем
   - Поиск пищи при голоде
   - Спасение от врагов и опасности
   - Поиск кровати для восстановления

4. ⛏️ УМНАЯ ДОБЫЧА
   - Приоритет: дерево → камень → уголь → железо
   - Автоматический выбор лучшего инструмента
   - Заполнение инвентаря стратегически

5. 🌍 ИССЛЕДОВАНИЕ
   - Процедурное исследование мира
   - Запоминание интересных мест
   - Поиск структур (деревни, храмы, шахты)
   - Отслеживание разных направлений

6. 🧠 СИСТЕМА ПАМЯТИ
   - Запоминает локации и ресурсы
   - Отслеживает угрозы и врагов
   - Помнит взаимодействия с игроками
   - История событий (до 100 последних)

7. 🛡️ БОЕВОЙ AI
   - Атакует враждебных мобов
   - Убегает при низком здоровье
   - Защищает владельца
   - Охотится на животных для еды

8. 🏗️ СТРАТЕГИЧЕСКОЕ ПЛАНИРОВАНИЕ
   - Выбирает оптимальную стратегию выживания
   - Переключается между режимами (выживание, добыча, исследование, охота)
   - Учитывает состояние инвентаря и здоровья

9. 🎮 КРАФТИНГ И ИНСТРУМЕНТЫ
   - Создаёт нужные инструменты
   - Управляет рецептами
   - Сбалансирован уровень инструментов

10. 🚗 ТРАНСПОРТ
    - Входит в лодки для быстрого передвижения
    - Использует животных

## КОМАНДЫ ДЛЯ ИГРОКА

БАЗОВЫЕ:
  "иди за мной"          → Бот следит за вами
  "стой"                 → Прекратить все действия
  "статус"               → Показать здоровье и голод
  "помощь"               → Список команд

ДОБЫЧА:
  "рубь деревья"         → Рубить древесину
  "копай камень"         → Добывать камень
  "копай уголь"          → Добывать уголь
  "копай железо"         → Добывать железо
  "копай алмазы"         → Добывать алмазы

ДЕЙСТВИЯ:
  "атакуй [враг]"        → Атаковать моба
  "собери предметы"      → Подобрать дроп
  "покажи инвентарь"     → Список предметов
  "спи"                  → Лечь спать в кровать
  "встань"               → Проснуться
  "исследуй"             → Начать исследование

СПЕЦИАЛЬНОЕ:
  "копай тоннель"        → Начать копать тоннель вперёд
  "прекрати копать"      → Остановить тоннель

## СИСТЕМА ПРИОРИТЕТОВ

Бот автоматически выбирает приоритет:

1️⃣ ВЫЖИВАНИЕ (если здоровье < 10)
   - Ищет кровать
   - Ест еду
   - Убегает от враагов

2️⃣ ГОЛОД (если еды < 10)
   - Ищет еду в инвентаре
   - Охотится на животных
   - Готовит мясо

3️⃣ ВРАГИ (если враги недалеко)
   - Атакует агрессивных мобов
   - Убегает если слишком опасно
   - Защищает себя

4️⃣ РЕСУРСЫ (если инвентарь не полный)
   - Добывает дерево, камень, уголь
   - Приоритет по нужности
   - Эффективная добыча

5️⃣ ИССЛЕДОВАНИЕ (если всё нормально)
   - Изучает новые территории
   - Ищет интересные структуры
   - Запоминает места

## ХАРАКТЕР БОТА

✅ Позитивные черты:
- Остроумный и смешной
- Отзывчивый на команды
- Старается выжить
- Помогает игроку

⚠️ Отрицательные черты:
- Саркастичный
- Иногда агрессивный
- Матерится (реально)
- Дерзкий в чате

Бот говорит только на русском языке!

## УСТАНОВКА И ЗАПУСК

1. Убедитесь, что установлен:
   - Node.js 14+
   - Minecraft сервер (локальный или удалённый)
   - Ollama + gemma2 модель

2. Установите зависимости:
   npm install

3. Создайте .env файл:
   MC_HOST=localhost
   MC_PORT=25565
   MC_USERNAME=BotName
   MC_VERSION= ВАША ВЕРСИЯ

4. Запустите:
   node bot.js

## ОТЛАДКА

Когда бот заходит в кричащий режим:
- Проверьте логи (console)
- Убедитесь, что Ollama работает
- Перезагрузите бота

Если бот не двигается:
- Проверьте координаты спавна
- Убедитесь, что нет препятствий
- Проверьте стека памяти
