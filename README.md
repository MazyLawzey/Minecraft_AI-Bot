# Простой Minecraft-бот на mineflayer.  
Проект чисто по приколу, но бот уже умеет делать полезные штуки и более-менее понимать, что ему пишут.

## Что умеет

- Ходить за игроком.
- Останавливаться.
- Рубить деревья.
- Подбирать предметы.
- Показывать инвентарь.
- Выбрасывать вещи.
- Атаковать мобов и животных.
- Ложиться спать.
- Садиться в лодку и вылезать из неё.
- Добывать некоторые ресурсы.
- Понимать обычные фразы через локальную ИИ-модель Ollama. [cite:165][cite:166]

## Как это устроено

Бот написан на `mineflayer`.  
Для "мозгов" используется локальная модель через Ollama, например `llama3.1:8b` на порту `11434`. [cite:165]

Проект разбит на модули, чтобы не держать весь ад в одном файле:

- `speech` — приветствие и базовый текст.
- `follow` — следование за игроком.
- `mine` — рубка деревьев.
- `inventory` — работа с инвентарём.
- `attack` — атака мобов и животных.
- `sleep` — сон.
- `vehicle` — лодка и прочее.
- `gather` — добыча ресурсов.
- `tunnel` — копание тоннеля.
- `ai` — понимание текста через локальную модель.
- `commands` — обработка команд и вызов модулей. [cite:165]
